not {set}R Documentation

Get Elements only Existed in Dataset a

Description

Get elements only existed in dataset a.

Usage

not(...)

a %not% b

Arguments

...

one or more objects

a

one object

b

the other object

Value

elements only existed in dataset a

Examples

A <- c("a","b")
B <- c("a","b","c","d")
not(B, A)

E <- c('d')
not(B, A, E)
A <- c("a","b","c")
B <- c("a","b","c","d")
B %not% A
