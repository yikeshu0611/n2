Combination of Characters or Vectors

Description

Combination of characters or vectors.

Usage

combination(...)

Arguments

...

one or more vectors

Value

binary combination

Examples

A <- c("a","b","c")
combination(A)

B <- c("a","b","c","d")
C <- c("a","e","h")
D <- c("a","b","e")
E <- c("a","c")
combination(A,B)
combination(A,B,C)
combination(A,B,C,D)
combination(A,B,C,D,E)
